# Dorn's Pose Controls v0.1.0

Direct pose keybinds for GAMMA / Anomaly.

## Behavior

- **Crouch key** (game bind) always crouches, never prone
- **Walk/Low Crouch key** (game bind) always prone, never walk or crouch
- **Sprint key** unchanged (stand + sprint)
- **Walk key** (MCM bind) is the only way to walk

Press the same pose key again to stand. Crouch and prone keys persist until another pose key is used (release is ignored). Walk key follows your game walk toggle/hold setting.

Compatible with Dorns_Prone_Fix — engine move states stay accurate.

## Installation

1. Install via MO2 like other Dorn mods
2. Enable in MCM and bind the Walk key

# Dorn's Pose Controls v0.1.0

Direct pose keybinds for GAMMA / Anomaly.

## Behavior

- **Crouch key** (game bind) always crouches, never prone
- **Walk/Low Crouch key** (game bind) always prone; press again to stand
- **Sprint key** unchanged (stand + sprint)
- **Walk key** (MCM bind) is the only way to walk

Crouch and walk follow your game toggle/hold settings. Compatible with Dorns_Prone_Fix.

## Installation

1. Download the latest release from https://github.com/JoshuaCarter/GAMMA-Pose-Controls/releases
2. Install via MO2 like normal
3. Enable in MCM and bind the Walk key
